# VIBEMed

VIBEMed (Versatile Intelligent Behavior-Evolving Medical framework) is an end-to-end [[wiki/concepts/multi-agent-system]] built for [[wiki/concepts/clinical-decision-support]]. It operates under the paradigm of [[vibe-medicine]], featuring three core agents: the Clinical Diagnostic Agent (CDA), the Therapeutic Execution Agent (TEA), and the Clinical Evolution Management Agent (CEMA). Its defining feature is a [[wiki/concepts/three-level-self-evolution]] mechanism (memory, model, and code) that continuously distills knowledge from clinical interactions to improve reasoning and treatment planning without manual developer intervention.

Source: [[phd2026toward|Toward Vibe Medicine: A Self-Evolving Multi-Agent Framework for Clinical Decision Support]]
