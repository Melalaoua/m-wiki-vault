# Vibecoding

[[vibecoding]] refers to an AI-assisted development approach where code generation and AI tools are heavily integrated directly into the codebase environment. For teams like [[kibu]], it has proven faster and more flexible than using traditional drag-and-drop web builders like [[wiki/entities/framer]].
