# Sketched-Isotropic-Gaussian Regularizer

Often abbreviated as SIGReg, an anti-collapse regularization term used in machine learning (specifically in JEPAs) that promotes feature diversity in embedding spaces. It encourages latent embeddings to match an isotropic Gaussian target distribution via multiple random 1D projections and univariate normality tests.
