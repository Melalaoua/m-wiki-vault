# Personal

This page serves as a top-level map connecting high-level life areas and projects within the personal domain. It encompasses both financial initiatives and open-source hardware development. Notable projects include the design and deployment of the [[personal/wiki/concepts/talis-ai-agent|Talis]] Discord bot, which automates budget tracking, systematizes personal investing, and calculates [[personal/wiki/concepts/fire-financial-independence-retire-early|FIRE]] metrics to support financial maturity ([[personal/wiki/sources/personal2026talisprd|TALIS-PRD]]). Alongside financial tooling, the domain tracks hardware endeavors like [[OOMWOO]], an open-source [[Robot Vacuum]] created by [[Maker's Pet]] using [[ROS 2]] ([[personal/wiki/sources/makerspet2026building|Building an Open-Source Robot Vacuum — Meet OOMWOO]]).
