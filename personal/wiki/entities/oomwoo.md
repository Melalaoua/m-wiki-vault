# OOMWOO

[[OOMWOO]] is an open-source, build-it-yourself [[Robot Vacuum]] created by [[Maker's Pet]]. It features a 3D-printable chassis, local-first operation with no cloud requirement, and native integration with [[Home Assistant]]. Its tech stack includes 2D LiDAR and [[ROS 2]] running on a [[Raspberry Pi]] and [[ESP32]]. The name is a rotational ambigram, meaning it reads the same when rotated 180 degrees.
