# Talis AI Agent

Talis is a custom, personal AI agent built as a Discord bot designed to automate personal finance management. It handles email ingestion to track transactions, monitors budgets, manages [[personal/wiki/concepts/fire-financial-independence-retire-early|FIRE]] metrics, and provides advisory insights for [[personal/wiki/concepts/pea-investing|PEA]] stock portfolios. Talis operates securely via a [[personal/wiki/concepts/hexagonal-architecture|Hexagonal architecture]] embedded with the [[mastra|Mastra agent framework]], ensuring read-only capabilities and manual trade execution.
