# Message Passing Neural Network

A [[Message Passing Neural Network]] (MPNN) is a generalized framework for [[Graph Neural Network]] architectures. It operates in three steps: gathering messages from neighboring nodes or edges, aggregating them using a permutation-invariant function (like sum or max), and applying an update function to form the new representation.
