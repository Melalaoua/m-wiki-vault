# ROS 2

Robot Operating System 2 (ROS 2) is a set of software libraries and tools that help build robot applications. It provides hardware abstraction, device drivers, and navigation stacks (like Nav2). It is widely used in both industrial robotics and open-source maker projects, such as the [[OOMWOO]] [[Robot Vacuum]].
