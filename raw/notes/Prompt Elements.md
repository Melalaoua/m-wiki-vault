On va couvrir plus d'exemple et d'applications, on remarquera que les élément suivants qui constitue le prompt :

- **Instruction** - une tâche spécifique ou une instruction qu'on veut que le modèle réalise.
  
- **Contexte** - informations externes ou contexte additionnel qui permet d'orienter le modèle dans le bonne direction.
  
-  **Input Data** - La question ou les données sur lequel on veut avoir des réponses.
  
- **Output indicator** - Le type de format ou d'output qu'ont veut.
  
On a pas besoin d'avoir les 4 éléments pour générer un prompt, le format dépend de la tâche donnée. On va monter des exemples plus concrets par la suite.
