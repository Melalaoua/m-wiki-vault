Le prompt engineering permet de mieux comprendre les capacités et les limites des Large language models (LLMs).

Les chercheurs utilise le prompt engineering dans l'idée d'améliorer les capacités des LLM dans une vaste gamme de tâches complexes et/ou communes comme par exemple le calcul arithmétique. 

Les développeurs utilisent le prompt engineering pour design des systèmes de prompts robustes et efficaces.

Toutefois, le prompt engineering, ce n'est pas juste la confection et le développement de prompts. C'est un domaine qui englobe un large éventail de capacités et de techniques qui sont utiles pour interagir avec les LLMs. C'est une capacité importante à développer, à utiliser et à comprendre pour saisir la profondeur des LLMs.

Il est possible, et fortement recommandé, d'utiliser le prompt engineering pour accroître l'efficacité des LLMs, en interface avec d'autres outils.
