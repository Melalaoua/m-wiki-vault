LangChain est un framework conçu pour simplifier la création d'applications qui utilisent les LLMs (comme l'api d'openAI). Ecrit en python et javascript, c'est un projet open-source lancé en octobre 2022 par Harrison Chase, un développeur travaillant pour la start up Robust Intelligence dans le secteur machine learning.

- **Facile à utiliser** : LangChain simplifie l'integration des LLM dans les applications. Ce qui rend la tâche plus facile pour les développeurs de créer, déployer des applications IA. Parmis les exemple d'applications IA on retrouve : l'analyse de document, la résumé de textes, la fonctionnalité chatbot.
- **Flexible** : LangChain inclus l'integration avec de multiples services ou systemes comme Amazone, Google, Microzoft Azure cloud storage, API wrappers pour des news, information sur des films, météo, et supporte de multiple sous-système de webscraping et des templates.
- **Mise en chaîne** : C'est la fonction mère de Langchain, il est possible de créer des chaînes qui sont des séquences d'opérations opérant à partir d'une seule input. L'output d'une opération sert d'input pour la suivante.
- **Chains customisables** : Bien que Langchain propose de multiple chaines déjà définies. Il est possible de créer des chaînes personalisées pour des besoins spécifiques.
- **Debugging et Mode Verbose** : Debugguer Langchain peut être facilement mis en place par l'activation du mode Verbose qui relate les états internes de la chaine pendant son execution, permettant le debugging.

### Pourquoi utiliser Langchain ?
Langchain est une librarie python qui permettre d'interagir plus facilement avec les LLMs comme GPT et de connecter des données externes, ou des applications.
