# Scratch

The lawless dump (ADR-0020). Fed from Obsidian directly and from the
`#scratch` Discord door, where every message lands verbatim in a dated file,
committed, and acknowledged with an emoji — no report, no routing, no LLM
call. Exempt from lint, sweep, consolidation, and both indexes. Graduating an
item into a brain happens only on the user's say-so, via a Filing.
