# Bridge

The user's cross-domain connection layer (ADR-0019). Notes here wikilink into
both `personal/` and `phd/` — the agent reads and cites them, labeled as the
user's own, but never writes, lints, sweeps, indexes, or stubs anything here.
Deliberately unmaintained by the machine. A note here that should become
maintained knowledge is filed into one brain on the user's request.
