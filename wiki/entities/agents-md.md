# AGENTS.md

`AGENTS.md` is a markdown file placed in the root of a software repository to provide essential context for [[wiki/concepts/ai-agent|AI coding agents]]. It is a high-impact, low-effort technique in [[wiki/concepts/harness-engineering|harness engineering]].

The purpose of this file is to bridge the gap of implicit knowledge and conventions that human developers share but agents do not. It typically contains:
- Project's technology stack and specific versions (e.g., Python 3.11, SQLAlchemy 2.0).
- Key architectural conventions and patterns.
- Instructions for setting up the development environment.
- Specific commands for running tests, linters, and other verification steps that define the project's [[wiki/concepts/definition-of-done|Definition of Done]].
