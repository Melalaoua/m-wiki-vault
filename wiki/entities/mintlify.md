# Mintlify

[[wiki/entities/mintlify]] is a documentation platform. Its 2025 marketing site featured an acclaimed top-fold design, decisive color usage, and a minimal component library. The site was so effective that [[wiki/entities/kibu]] used it as a North Star and literally rebuilt it pixel-by-pixel to bootstrap their own web presence.
