# Harness-Induced Failure

A [[wiki/concepts/harness-induced-failure|harness-induced failure]] occurs when an [[wiki/concepts/ai-agent|AI agent]] fails to complete a task correctly, not because the model lacks the necessary capability, but because the execution environment has structural defects. 

This highlights the [[wiki/concepts/capability-gap|capability gap]] between benchmark performance and real-world application. The failure is caused by issues in the [[wiki/concepts/harness|harness]], such as vague requirements, missing context, or a lack of verification methods. The solution is to apply [[wiki/concepts/harness-engineering|harness engineering]] principles to diagnose and fix the environmental problem rather than assuming the model is at fault.
